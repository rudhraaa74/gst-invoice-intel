# Dataset Information

The data for this repository is not stored in source control to keep the repository lightweight and to protect potentially sensitive information.

**How to get the data:**
- **WhatsApp:** The dataset ZIP file has been shared in our project WhatsApp group. Please download it from there.
- **Kaggle:** Alternatively, you can download the dataset from your respective Kaggle accounts where it has been shared with you.

Once downloaded, extract the contents and place all raw data files into the `/data/raw` directory.
