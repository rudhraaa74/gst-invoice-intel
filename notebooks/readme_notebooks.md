# Notebooks Workflow

To ensure smooth collaboration and leverage cloud compute resources efficiently, please follow this workflow when working with notebooks in this repository:

1. **Write and Edit Locally**: Write and edit your code locally in your preferred code editor (e.g., VS Code, Jupyter, etc.).
2. **Copy to Kaggle**: Copy and paste your code into a Kaggle notebook.
3. **Run on Kaggle**: Execute your code on Kaggle, utilizing their 2x T4 GPUs (or other hardware accelerators) for training and heavy processing.
4. **Export and Commit**: Once your work is complete, export the final `.ipynb` file from Kaggle and push it back to this repository under the `/notebooks` directory.
